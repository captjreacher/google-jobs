# google-jobs

Local web-design lead engine — find independent businesses that already have great Google reviews but weak or missing websites, then build them modern concept sites packaged as a single click-through pitch board.

## West Auckland board

`westside.html` is a self-contained operator dashboard profiling five real West Auckland businesses (all >= 4.2 stars on Google) whose web presence is missing or dated. Each card shows the business's real current state (an actual screenshot, or a "No website" tile) and links to a distinct modern concept-site rebuild. The Swanson Worx rebuild includes a working speed-to-quote & booking demo.

Open `westside.html` in any browser. Hero and screenshot images load from public URLs, so an internet connection is needed to render them.
